## I want to merge multiple images into one PDF file

1. Select all images
2. Right-click and select ```Print```
On windows 11, print option may be hided, therefore, we should select ```Show more option``` then ```Print```.

## On MS Word, I want to format multiple object (shapes, ...) to the same size?

Simply select all and then goto ```Format```, then config the size (width and height)
